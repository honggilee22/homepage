통합 모듈 상세 기능

8개 모듈이 하나로 연결된 완벽한 워크플로우

1

블로그 분석

활성화

RSS와 네이버 방문자 API로 블로그 현재 상태를 정밀 진단

주요 기능

  • RSS 기반 전체 포스트 수집 및 키워드 추출
  • 방문자 데이터 실시간 수집 (일별/주별/월별)
  • 블덱스 점수 및 등급 자동 산출
  • 키워드별 노출/클릭률 분석
  • 경쟁 블로그 대비 성과 비교
블로그 분석 화면 스크린샷
2

키워드 생성

활성화

Gemini AI 활용 키워드 팩 생성과 사용자 선택 하이브리드 방식

주요 기능

  • AI 기반 연관 키워드 자동 생성
  • 검색량 vs 문서수 비율 분석
  • 월 발행량 기반 경쟁도 측정
  • 롱테일 키워드 자동 조합
  • 추천도 % 표시 (95%, 88% 등)
키워드 생성 화면 스크린샷
3

콘텐츠 준비

준비됨

사용자 프로필과 프롬프트 중앙 관리로 일관된 톤앤매너 유지

주요 기능

  • 사용자 프로필 중앙 관리 (SharedDataManager)
  • 프롬프트 플레이스홀더 자동 치환
  • 수동 글쓰기 모드 (제목/본문 직접 입력)
  • AI 추천 제목/키워드/니즈 제안
  • 자동/수동 모드 전환
4

통합 콘텐츠 생성

활성화

Claude AI와 Gemini 브라우저 직접 연동으로 API 비용 제로

주요 기능

  • Claude AI 브라우저 직접 연동 (API 비용 무료)
  • Gemini 스타일링 적용
  • HTML 태그 자동 적용 및 포맷팅
  • Word(DOCX) 자동 내보내기
  • 자동화 모드 감지 및 전체 시퀀스 실행
통합 생성 화면 스크린샷
5

이미지 자동 생성

활성화

6개 브라우저 탭 동시 실행으로 대량 이미지 고속 생성

주요 기능

  • 6개 탭 병렬 처리로 생성 속도 극대화
  • 프롬프트 자동 주입 및 실행
  • 타임아웃 및 재시도 로직
  • 이미지 경로 중앙 관리
  • 실패 시 대체 이미지 자동 저장
이미지 생성 화면 스크린샷
6

이미지 편집

활성화

6장 동시 편집 + 10장 순서 편집기로 완벽한 이미지 관리

이미지 순서 편집기

  • 최대 10장 이미지 드래그앤드롭 재정렬
  • 인덱스 배지 및 파일명 표시
  • 이미지별 설명 텍스트 입력
  • 정렬된 경로와 설명을 다음 모듈로 전달

편집 기능

  • 텍스트/박스 오버레이 추가
  • 워터마크 일괄 적용
  • 네이버 스타일 컬러세트 제공
  • 레이어 관리 및 자동화 모드
이미지 편집 화면 스크린샷
7

블로그 자동 발행

준비됨

PyAutoGUI 휴먼라이크 패턴으로 안전한 자동 발행

주요 기능

  • 브라우저 자동 실행 및 활성화
  • 작성 페이지 자동 이동
  • 발행 버튼 자동 클릭
  • 예약 시간 자동 설정
  • 발행 URL 자동 복사 및 저장
  • 실패 시 백업 루트 자동 실행
8

인스타그램 발행

준비중

블로그 콘텐츠를 인스타그램용으로 자동 변환 발행

예정 기능

  • 블로그 본문 → 인스타 캡션 자동 변환
  • 이미지 1:1 비율 자동 리사이징
  • 해시태그 자동 생성
  • 스토리/릴스 템플릿 제공
